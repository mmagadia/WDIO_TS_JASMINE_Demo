# Introduction

Sample test suite using WebdriverIO, Typescript and Jasmine.

## Angular 7 Test site

https://github.com/mmagadia/AngularApp
See instructions for running locally.

## Run webApp locally

npm start

## Installation

npm install

npm test wdio.conf.js
